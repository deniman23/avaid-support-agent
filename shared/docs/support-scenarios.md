## Маршрутизация

1. Определить категорию по сообщению пользователя.
2. Retrieval из **Avaid Rules** (приоритет), при нехватке — **Avaid Codebase**.
3. Фактические проверки — только через tools db-api (не выдумывать данные).
4. Тарифы и возврат денег — справочно + эскалация в поддержку.

# Акции Ozon и ярлыки заказов

---

## Акции Ozon — исключение товаров

### Что это

Ozon периодически добавляет товары в акции и автоматически снижает на них цены. Если вы не хотите, чтобы Avaid участвовал в этом — конкретный товар можно исключить из акций.

### Как работает

Переключатель **«Удалять из акций»** в таблице товаров (раздел **«Цены и остатки»**). Когда он включён:

1. Avaid получает список текущих акций Ozon
2. Для каждой акции отправляет запрос на деактивацию товара
3. Товар убирается из всех активных акций Ozon

Синхронизация акций запускается автоматически при каждом цикле обновления Ozon.

**Важно:** Исключение из акций активно работает только для Ozon. Для WB, Яндекс и Avito включённый переключатель **«Удалять из акций»** запрещает отправку цены по этому товару — т.е. цена просто не обновляется.

### Массовое исключение

В разделе товаров есть массовое действие «Исключить из акций» — применяется сразу к нескольким выбранным товарам.

## Ярлыки и штрихкоды заказов

### Что это

Avaid умеет генерировать PDF с ярлыками (баркодами) для заказов. Ярлык печатается и клеится на отправление — маркетплейс сканирует его при приёмке.

### Поддерживаемые маркетплейсы

Генерация ярлыков работает для: **Ozon, Wildberries, Яндекс.Маркет, Avito**.

### Режимы печати

**Все заказы** — PDF со всеми актуальными заказами магазина.

**Заказы на сегодня/завтра** — только заказы с датой отгрузки сегодня или завтра (по московскому времени). Удобно для ежедневной печати перед отправкой.

**По выбранным заказам** — указываете конкретные ID заказов.

### Структура PDF

Avaid генерирует PDF с двумя частями:
1. Титульный лист-таблица в хронологическом порядке — список заказов для сверки
2. Страницы с баркодами — по одному ярлыку на заказ

### Отправка в Telegram

Ярлыки можно получать автоматически в Telegram-чат магазина. В настройках магазина → вкладка **«Уведомления»** → раздел **«Баркод»** — там настраивается **«Выгрузка в Telegram»**, **«Вид ленты»** и **«Времена отправки»**. Avaid отправляет PDF по расписанию или при появлении новых заказов, готовых к отгрузке.

## Часто задаваемые вопросы

**В: Включил «Удалять из акций» для товара, но Ozon всё равно добавляет его в акцию.**
О: Исключение обрабатывается при следующей синхронизации акций. Если проблема повторяется — возможно, Ozon добавляет товар в новую акцию после исключения. Avaid исключает при каждом цикле синхронизации.

**В: Как скачать ярлыки для сегодняшних заказов?**
О: В разделе «Ярлыки» выберите режим «Заказы на сегодня/завтра» — скачается PDF только с актуальными отправлениями.

**В: Ярлыки приходят в Telegram, но некоторых заказов нет в PDF.**
О: В PDF попадают только заказы с датой отгрузки сегодня или завтра (если включён соответствующий режим). Заказы без даты отгрузки или с другой датой не включаются.

**В: Для каких маркетплейсов работает исключение из акций?**
О: Активное исключение (через API Ozon) — только для Ozon. Для остальных маркетплейсов включённый переключатель **«Удалять из акций»** просто блокирует обновление цены по этому товару.

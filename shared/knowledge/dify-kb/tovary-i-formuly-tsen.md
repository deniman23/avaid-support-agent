# Товары на маркетплейсах — управление и формулы цен

## Что это

Раздел **«Цены и остатки»** — это список всех товаров ваших магазинов в Avaid. Здесь можно управлять привязкой к МойСклад, включать/выключать синхронизацию, задавать формулы цен и скидки.

---

## Управление синхронизацией товара

Для каждого товара отдельно можно включать и выключать через колонки в таблице:

- **«Синхронизация остатков»** — отправлять ли остатки этого товара на маркетплейс
- **«Синхронизация цен»** — отправлять ли цену этого товара на маркетплейс
- **«Удалять из акций»** — исключить товар из акций (цена не меняется при акционных обновлениях)

Если для магазина в целом включена синхронизация, но у конкретного товара выключена — по этому товару данные не отправляются.

---

## Привязка и отвязка товара от МойСклад

**Привязка вручную** — в разделе «Непривязанные» выбираете товар с маркетплейса и указываете, какому товару МС он соответствует.

**Массовая привязка** — можно привязать несколько товаров сразу.

**Отвязка** — убирает связь между товаром маркетплейса и товаром МС. После отвязки остатки и цены из МС для этого товара перестают синхронизироваться.

После привязки Avaid автоматически обновляет остатки из МС для привязанных товаров.

---

## Формулы цен

Формула позволяет автоматически рассчитывать цену для маркетплейса на основе данных из МойСклад.

### Как работает

В карточке товара в поле **«Формула цены»** задаётся математическое выражение. Поддерживаются операции `+ - * /` и скобки.

Доступные переменные в формуле:
- `price` — текущая цена товара на маркетплейсе
- `min_price` — минимальная цена
- `ms_price` — цена из МойСклад (по уровню цены или цена продажи)

**Пример формул:**
- `ms_price * 1.15` — цена МС + 15% наценка
- `ms_price + 500` — цена МС + 500 рублей
- `23710` — фиксированная цена (просто число тоже работает как формула)

### Важно про фиксированную цену
Если вы хотите задать фиксированную цену, укажите число в поле формулы — например `23710`. Без формулы рассчитанная цена не используется и уходит старая цена с маркетплейса.

### Пересчёт цены

После сохранения формулы Avaid сразу рассчитывает итоговую цену и ставит задачу отправки цены на маркетплейс.

Принудительно пересчитать цену без изменения формулы — кнопка **«Пересчитать цену»** в карточке товара.

---

## Ценовые уровни МойСклад на уровне товара

Если в МС несколько уровней цен (оптовая, розничная и т.д.), можно указать для конкретного товара, какой уровень использовать:

- **Не указан** → используется уровень, заданный для магазина в **«Уровень цены по умолчанию»**
- **Указан** → берётся цена именно этого уровня из МС

Приоритет: уровень товара → уровень магазина → цена продажи МС.

---

## Скидка

Для каждого товара можно задать скидку в процентах (0–99%). Скидка отправляется на маркетплейс вместе с ценой. Это поле — именно скидка для маркетплейса, не скидка в МС.

---

## Часто задаваемые вопросы

**В: Задал формулу, но цена на маркетплейсе не изменилась.**
О: Проверьте: 1) включена ли **«Синхронизация цен»** для этого товара, 2) включена ли **«Синхронизация цен»** для магазина в целом (настройки магазина), 3) для WB — заполнен ли **«Токен для цен WB»** в настройках магазина.

**В: Хочу задать фиксированную цену 5000 рублей, не зависящую от МС.**
О: Введите `5000` в поле формулы. Число без переменных — это фиксированная цена.

**В: Пересчитал цену по формуле, цена обновилась, но на маркетплейс не ушла.**
О: После пересчёта Avaid ставит задачу отправки в очередь — она выполнится через несколько минут. Если долго не обновляется — проверьте, нет ли ошибок с подключением к маркетплейсу.

**В: Отвязал товар от МС. Остатки на маркетплейсе обнулились.**
О: После отвязки Avaid больше не отправляет остатки. Маркетплейс оставляет то значение, которое было последним. Для обнуления нужно явно отправить 0 — это не происходит автоматически.

**В: Хочу разные цены на разных маркетплейсах для одного товара МС.**
О: Для каждого магазина (даже одного товара на разных маркетплейсах) формула задаётся отдельно в карточке товара этого магазина.

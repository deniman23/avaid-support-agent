# Подключение магазинов

## Поддерживаемые маркетплейсы

Avaid поддерживает: **Ozon, Wildberries, Яндекс.Маркет, Avito, Лемана ПРО**.

Для каждого маркетплейса нужны свои ключи доступа — у них разные системы авторизации.

---

## Ozon

**Что нужно:** Client ID и API Key.

**Где взять:** Личный кабинет Ozon → Настройки → API ключи.

**Важно:** API Key должен быть создан с правами администратора или с полным набором прав. Ключи с ограниченными правами (только статистика или только цены) не подойдут — Avaid не сможет читать товары, остатки и заказы.

**Проверка:** При сохранении Avaid автоматически проверяет ключи — делает тестовый запрос к API Ozon. Если ключ принят, но прав не хватает — будет сообщение «у ключа не хватает прав, создайте ключ в режиме администратора».

---

## Wildberries

**Что нужно:** API Key (основной токен для заказов/товаров/остатков) и отдельный токен для цен.

**Где взять:** Личный кабинет WB → Настройки → Доступ к API.

**Важно:** WB разделяет токены по назначению:
- Основной токен — для синхронизации товаров, заказов, остатков.
- **Токен для цен WB** — отдельный токен, нужен для обновления цен. Без него цены на WB не обновятся даже при включённой **«Синхронизации цен»**.

**Схемы работы на WB:** Avaid поддерживает FBS (склад продавца), DBW/DBS (доставка продавца), FBW (склад WB). Для каждой схемы — своя очередь заказов.

---

## Яндекс.Маркет

**Что нужно:** API Key, Campaign ID (ID кампании), Business ID (ID бизнес-аккаунта).

**Где взять:** Кабинет Яндекс.Маркета → Настройки → API.

**Особенность:** В настройках магазина → вкладка **«Цены»** есть поле **«Режим синхронизации цен»**: обновлять цены в кабинете Яндекса, в магазине, или оба варианта одновременно.

---

## Avito

**Что нужно:** Client ID и Client Secret (в поле API Key вводится Client Secret).

**Где взять:** Avito для бизнеса → Настройки → Авторизация → OAuth приложения.

**Как работает:** Avito использует OAuth2, Avaid получает токен автоматически при каждом запросе.

**Особенность привязки:** Для автопривязки товаров Avito к МойСклад нужен специальный атрибут. В карточке товара МС должен быть атрибут с именем «Avito», в котором записан ID объявления с Avito. Без этого автопривязка по Avito не работает.

---

## Лемана ПРО (Леруа Мерлен)

**Что нужно:** Client ID и Client Secret.

**Где взять:** Личный кабинет партнёра Лемана ПРО → API настройки.

**Как работает:** Лемана использует OAuth2, Avaid получает токен автоматически.

---

## Как Avaid проверяет ключи при подключении

При сохранении магазина Avaid делает тестовый запрос к API маркетплейса и показывает результат:

- **Ключи прошли проверку** — всё в порядке, синхронизация начнётся.
- **Ошибка авторизации** — ключ неверный или истёк. Нужно создать новый.
- **Не хватает прав** (Ozon) — ключ работает, но у него ограниченные права. Нужен ключ с правами администратора.
- **Временная ошибка на стороне маркетплейса** — API маркетплейса недоступен. Попробуйте позже.

---

## Доступность маркетплейсов по тарифу

Не все маркетплейсы доступны на всех тарифах. Если при попытке добавить магазин появляется сообщение об ограничении — проверьте ваш тариф и доступные маркетплейсы.

---

## Часто задаваемые вопросы

**В: Ключи ввёл, Avaid говорит «ошибка авторизации». Что не так?**
О: Проверьте, что скопировали ключ полностью (без лишних пробелов). Для Ozon — убедитесь, что указали и Client ID, и API Key (нужны оба). Для WB — основной API Key и отдельный **«Токен для цен WB»** — это разные поля.

**В: Ozon ключ принят, но написано «не хватает прав».**
О: Ваш API Key создан с ограниченными правами. Зайдите в кабинет Ozon → API ключи → создайте новый с правами администратора или добавьте нужные разрешения текущему.

**В: Подключил WB, цены не обновляются.**
О: Для обновления цен на WB нужен отдельный **«Токен для цен WB»**. Проверьте в настройках магазина → вкладка **«Цены»** — заполнено ли это поле.

**В: Для Avito что вводить в поле «API Key»?**
О: В поле API Key вводится Client Secret от вашего OAuth-приложения Avito. Client ID вводится в отдельное поле.

**В: Можно подключить несколько магазинов одного маркетплейса?**
О: Да, каждый магазин добавляется отдельно со своими ключами. Количество магазинов зависит от тарифа.

**В: Магазин подключён, но синхронизация не идёт.**
О: Убедитесь, что магазин активен (не выключен). Также проверьте подключение МойСклад — без него синхронизация заказов и документов не работает.

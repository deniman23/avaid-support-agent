# Ошибки синхронизации

Краткое описание возможностей Avaid (ветка dev, пакет `taskevent`).

## Бизнес-логика (из комментариев и кода backend-go)
- CreateRunning создаёт запись в task_events со статусом running и show_in_notifications = true.
- Возвращает ID созданной записи.
- MarkSuccess переводит задачу в success, проставляет completed_at и duration.
- MarkError переводит задачу в error, проставляет completed_at, duration и description.
- GetShopName возвращает название магазина по ID (для заголовка уведомления).

## Для агента поддержки
- Переводи логику кода в **шаги в интерфейсе Avaid**.
- Не цитируй имена функций пользователю.

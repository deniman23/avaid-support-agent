# Ozon — привязка (связка) товаров с МойСклад

## Вопросы пользователя

- Как происходит связка товаров на Ozon?
- Как привязать товары Ozon к МойСклад?
- Как сопоставить карточки Ozon с позициями в МС?

## Короткий ответ

Связка на Ozon — сопоставление **уже существующей** карточки на Ozon с позицией в **МойСклад**. В Avaid: раздел **«Цены и остатки»** → магазин Ozon → **«Автопривязка»** → **Запустить** (подбор по SKU, названию, штрихкоду). После связки Avaid передаёт остатки и цены из МойСклад на Ozon.

## Шаги в Avaid

1. **Цены и остатки** → выберите магазин **Ozon**.
2. **Автопривязка** → **Запустить** (подбор по SKU, названию, штрихкоду и др.).
3. Фильтр **«Только непривязанные товары»**.
4. Для оставшихся — **«Привязать»** → выберите товар/артикул из МойСклад.
5. При необходимости — **«Синхронизировать»** в каталоге.

К одной позиции МойСклад можно привязать несколько товаров с разных магазинов.

## Частые ошибки в понимании

- **Неверно:** Avaid автоматически создаёт новые карточки на Ozon из МойСклад.
- **Неверно:** «Обратитесь в поддержку» для ручной привязки — она делается в интерфейсе Avaid.
- **Верно:** карточки на Ozon уже должны существовать; Avaid их **связывает** с МС для синхронизации остатков и цен.

## Когда нужен email / tools

Только чтобы посмотреть **конкретные непривязанные товары** в аккаунте клиента. На общий вопрос «как работает связка» email не нужен.

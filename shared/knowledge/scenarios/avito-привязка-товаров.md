# Avito — привязка товаров

## Вопросы пользователя

- Как привязать товары Avito к МойСклад?
- Как работает автопривязка на Авито?

## Короткий ответ

Связка — сопоставление объявления Avito с позицией МойСклад. Для автопривязки нужно доп. поле **«Avito»** (строка) в карточке товара МС со значением **ID объявления** из Avito.

## Шаги в Avaid

1. В МойСклад: доп. поле **«Avito»** (тип: Строка) → ID из объявления Avito (под названием объявления).
2. **Цены и остатки** → магазин Avito → **Автопривязка** → параметр **«Avito / Avito»** → Запустить.
3. Остаток — **«Только непривязанные»** → **Привязать** вручную.

## Частые ошибки

- **Неверно:** связка по артикулу или названию вместо ID объявления для автопривязки Avito.
- **Неверно:** Avaid создаёт новые объявления на Avito — создание объявлений через приложение не обещается.

## Когда нужен email / tools

Для проверки конкретных непривязанных позиций в аккаунте клиента.

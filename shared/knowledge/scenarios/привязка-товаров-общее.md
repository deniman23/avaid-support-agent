# Привязка товаров (все маркетплейсы)

## Вопросы пользователя

- Как привязать товары к МойСклад?
- Как работает автопривязка?

## Короткий ответ

**Привязка** связывает уже существующий товар на маркетплейсе с позицией в **МойСклад** — после этого Avaid передаёт остатки и цены из МС на площадку.

**Автопривязка** в **«Цены и остатки»** — автоматический подбор пары товара маркетплейса с позицией **МойСклад** по **SKU, названию, штрихкоду** (общее описание; email не нужен).

## Шаги в Avaid

1. **Цены и остатки** → выберите **магазин** (Ozon, WB, Яндекс и т.д.).
2. **Автопривязка** → **Запустить** — система предложит совпадения по SKU, названию, штрихкоду.
3. Включите фильтр **«Только непривязанные товары»**.
4. Для оставшихся — **«Привязать»** → выберите позицию из МойСклад.
5. К одной позиции МС можно привязать несколько товаров с разных магазинов.

## Частые ошибки в понимании

- **Неверно:** автопривязка создаёт новые карточки на маркетплейсе из МойСклад.
- **Неверно:** для общего вопроса «как работает автопривязка» нужен email — email только чтобы посмотреть **ваши** непривязанные в аккаунте.
- **Верно:** карточки/объявления на площадке уже должны существовать; Avaid их **связывает** с МС.

Для **Avito** — отдельный сценарий (поле «Avito» и ID объявления).  
Для **Ozon** — отдельный сценарий (карточки уже на Ozon, не «создание из МС»).

## Когда нужен email / tools

Только для списка непривязанных в аккаунте клиента.

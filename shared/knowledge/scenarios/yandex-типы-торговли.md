# Яндекс.Маркет — типы торговли

## Вопросы пользователя

- Какие схемы продаж на Яндекс.Маркете?
- Какие схемы на Яндекс.Маркете?
- FBY, FBS, Экспресс — что выбрать?

## Короткий ответ

На **Яндекс.Маркете** в Avaid (это **не Ozon** и **не WB**): **FBY, FBS, FBS Экспресс, DBS** — каждая схема **отдельный магазин**. Схемы **FBO** здесь **нет** (FBO — это Ozon/WB).

## Шаги в Avaid

1. **Магазины** → **Добавить магазин** → **Yandex**.
2. Тип: FBY / FBS / FBS Экспресс / DBS.
3. Из ЛК: Campaign ID, Business ID, API Key (Настройки → API и модули).

**Экспресс** — отдельный магазин с пометкой «Экспресс» в названии и отдельным токеном.

## Частые ошибки в понимании

- **Неверно:** на Яндекс.Маркете в Avaid те же схемы, что на Ozon (только FBO/FBS) — здесь **FBY, FBS, FBS Экспресс, DBS**.
- **Неверно:** FBO на Яндекс — правильное название схемы склада маркетплейса здесь **FBY**.
- **Верно:** каждая схема — отдельный магазин в Avaid; тип при создании не меняется.

# Ozon — типы торговли (FBO / FBS)

## Вопросы пользователя

- Какие типы торговли на Ozon?
- Чем отличается FBO и FBS на Ozon?
- Какую схему выбрать при создании магазина?

## Короткий ответ

В Avaid для Ozon: **FBO** и **FBS**. Каждая схема — **отдельный магазин** в приложении. Тип выбирается при создании и **не меняется** потом.

## Шаги в Avaid

1. **Магазины** → **Добавить магазин** → маркет **Ozon**.
2. Выберите тип **FBO** или **FBS**.
3. Укажите **Client ID** и **API Key** из ЛК Ozon (Настройки → Seller API).

**FBO** — товары на складе Ozon, отгрузку и доставку ведёт маркетплейс.  
**FBS** — товары у продавца, доставку организует Ozon.

## Чем отличается FBO и FBS на Ozon

| | FBO | FBS |
|---|-----|-----|
| Где товар | На складе Ozon | У продавца |
| Кто доставляет | Ozon | Ozon забирает у продавца |
| В Avaid | **Отдельный магазин** | **Отдельный магазин** |

Тип схемы при создании магазина **не меняется** — для второй схемы нужен **новый** магазин.

## Частые ошибки в понимании

- **Неверно:** можно сменить FBO на FBS в том же магазине — нужен новый магазин с другим типом.
- **Верно:** для FBO и FBS создаются два отдельных магазина в Avaid.

## Когда нужен email / tools

Не требуется для объяснения типов. Email — только для проверки настроек конкретного магазина клиента.

# Как работает автопривязка

## Вопросы пользователя

- Как работает автопривязка?
- Что делает автопривязка в Avaid?
- Автоматическая привязка товаров — как устроена?

## Короткий ответ

**Автопривязка** в Avaid подбирает пару «товар на маркетплейсе ↔ позиция **МойСклад**» по **SKU, названию, штрихкоду** и другим полям. Запуск: **«Цены и остатки»** → выберите магазин → **«Автопривязка»** → **Запустить**. Email для этого не нужен.

## Шаги в Avaid

1. **Цены и остатки** → магазин (Ozon, WB, Яндекс и т.д.).
2. **Автопривязка** → **Запустить**.
3. Фильтр **«Только непривязанные товары»** — проверить предложенные пары.
4. Остатки без пары — **«Привязать»** вручную из каталога **МойСклад**.

## Частые ошибки в понимании

- **Неверно:** автопривязка создаёт новые карточки на маркетплейсе.
- **Неверно:** без email нельзя объяснить автопривязку — объяснение общее, email только для списка **ваших** непривязанных.
- **Верно:** товары на площадке уже существуют; Avaid сопоставляет их с **МойСклад**.

## Когда нужен email / tools

Не требуется для общего описания.

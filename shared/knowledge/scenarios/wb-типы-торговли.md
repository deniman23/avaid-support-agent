# Wildberries — типы торговли

## Вопросы пользователя

- Какие типы торговли на WB / Wildberries?
- Что такое FBO, FBS, DBS, DBW?

## Короткий ответ

В Avaid для Wildberries: **FBO, FBS, DBS, DBW**. Каждая схема — отдельный магазин.

## Шаги в Avaid

1. **Магазины** → **Добавить магазин** → **Wildberries**.
2. Выберите тип схемы.
3. Токен из ЛК WB: Профиль → Интеграция по API → Создать токен (для интеграции вручную, чтение и запись; без «Чат» и «Вопросы и отзывы»).

**FBO** — товары на складе WB, WB пакует и доставляет.  
**FBS** — товары у продавца, доставляет WB.  
**DBS** — хранение и доставка у продавца.  
**DBW** — хранение у продавца, сдача в пункт выдачи WB.

## Частые ошибки в понимании

- **Неверно:** на WB есть «Леруа», «Ситилинк» или розничные сети — в Avaid для WB только FBO, FBS, DBS, DBW.
- **Неверно:** один магазин в Avaid совмещает FBO и FBS — для каждой схемы создаётся **отдельный** магазин.
- **Верно:** FBO/FBS/DBS/DBW — это схемы логистики Wildberries, не названия других маркетплейсов.

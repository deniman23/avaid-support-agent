# Какие маркетплейсы поддерживает Avaid

## Вопросы пользователя

- Какие маркетплейсы можно подключить?
- С какими площадками работает Avaid?

## Короткий ответ

В Avaid можно подключить: **Wildberries, Ozon, Яндекс.Маркет, Авито.**  
Отдельно в тарифах — **Лемана ПРО** (B2B; это **не** розничный «Леруа Мерлен»).

**Не поддерживаются:** Amazon, Ситилинк, розничный Леруа Мерлен и другие площадки вне списка выше.

## Схемы по площадкам

| Маркетплейс | Схемы |
|-------------|--------|
| Wildberries | FBO, FBS, DBS, DBW |
| Ozon | FBO, FBS |
| Яндекс.Маркет | FBY, FBS, FBS Экспресс, DBS |
| Авито | ручная и автопривязка объявлений |

## Порядок подключения

**Магазины** → **Добавить магазин** → API-ключ → склад и рубильники → привязка товаров.

## Не поддерживаются

Леруа Мерлен (розница), Ситилинк, Amazon и др. вне списка выше — честно сообщить, что в Avaid их нет.

## Когда нужен email / tools

Не нужен для общего списка. Tools — только для магазинов **конкретного** аккаунта.

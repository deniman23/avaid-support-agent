# Связка (привязка) товаров Ozon в Avaid

## Что такое связка

**Связка** — это сопоставление **уже существующей** карточки товара на Ozon с позицией в МойСклад.  
После связки Avaid передаёт остатки и цены из МойСклад на Ozon.

На Ozon можно связывать обычные товары и комплекты.

## Как связать товары (пошагово)

1. Раздел **«Цены и остатки»**.
2. Выберите магазин **Ozon**.
3. **«Автопривязка»** → **«Запустить»** (подбор по SKU, названию, штрихкоду и др.).
4. Фильтр **«Только непривязанные товары»**.
5. Для оставшихся — **«Привязать»** → выбрать товар/артикул из МойСклад.
6. К одной позиции МойСклад можно привязать несколько товаров с разных магазинов.

При необходимости после изменений — **«Синхронизировать»** в каталоге.

## Что НЕПРАВИЛЬНО говорить клиенту (частые ошибки)

- **Неверно:** «Avaid автоматически создаёт карточки товаров на Ozon из МойСклад».
- **Неверно:** «Достаточно внести товары в МойСклад — и карточки на Ozon появятся сами».
- **Неверно:** «Для ручной привязки обратитесь в поддержку Avaid» — ручная привязка делается **в интерфейсе Avaid** (кнопка «Привязать»).
- **Неверно:** «Сначала создайте карточки в МойСклад, система сама выгрузит их на Ozon» — карточки на Ozon должны **уже быть** (личный кабинет Ozon / импорт), Avaid их **связывает** с МС для синхронизации остатков и цен.

## Если связка не сработала

- Проверьте, что магазин Ozon подключён в Avaid (тип FBO/FBS выбирается при создании и не меняется).
- В МойСклад заполнены артикул, штрихкод и другие поля для сопоставления.

## Уценка на Ozon

Ozon допускает только 1 единицу уценённого товара. Карточку уценки привязывают к основному товару МойСклад.

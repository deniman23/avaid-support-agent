# База знаний агента поддержки Avaid

## О продукте

Avaid — платформа для автоматизации продаж на маркетплейсах. Связывает МойСклад с Wildberries, Ozon, Яндекс.Маркет, Авито. Автоматически синхронизирует остатки, цены, заказы, возвраты. Экономит до 90% времени на рутинных операциях.

## Что умеет платформа

- Централизованное управление всеми магазинами в одном интерфейсе
- Автоматическая синхронизация остатков и цен на всех маркетплейсах
- Автоподтверждение новых заказов
- Создание документов в МойСклад (заказы, отгрузки, возвраты)
- Резервирование товаров при поступлении заказа
- Защита от нежелательных акций (особенно Ozon и WB)
- Telegram-уведомления о заказах, возвратах, акциях в реальном времени
- Печать этикеток и титульных листов прямо в Telegram
- Привязка товаров (автоматическая и ручная)
- Уценка Ozon (автоподнятие остатка после каждой продажи)
- Восстановление данных с любой даты

## Поддерживаемые маркетплейсы

- **Wildberries** — FBO, FBS, DBS, DBW
- **Ozon** — FBO, FBS
- **Яндекс.Маркет** — FBY, FBS, FBS Экспресс, DBS
- **Авито** — ручная и автопривязка

## Как подключить маркетплейс (общий порядок)

1. Раздел «Магазины» → «Добавить магазин»
2. Ввести название, выбрать маркет, указать тип схемы работы
3. Добавить API-ключ из личного кабинета маркетплейса
4. Подключить Telegram-уведомления
5. Настроить магазин (организация, склад, рубильники)
6. Привязать товары (автопривязка или вручную)
7. Активировать магазин → синхронизировать остатки и цены

## Важные рубильники в настройках магазина

- **Подтверждать новые заказы** — автоматически переводит заказ в "готов к отгрузке" за 10 минут
- **Создавать Заказы покупателя** — появление документа в МойСклад
- **Резервировать товары** — защита от двойной продажи
- **Создавать Отгрузки** — автоматическое списание со склада
- **Создавать Возвраты покупателя** — учёт возвратов
- **Удалять из акций** — защита от принудительных акций Ozon/WB

## Синхронизация остатков

Три режима склада:
- Один склад на все маркеты
- Сумма нескольких складов
- Отключено (ручное управление)

## Цены

- Уровень цены по умолчанию — из основного поля "Цена продажи" в МойСклад
- Можно настроить отдельные уровни цен для каждого маркета
- Для Яндекс.Маркет — отдельная цена для FBS и Экспресс или общая

## Telegram-уведомления

1. Создать Telegram-канал (публичный → после настройки сделать частным)
2. Добавить @AvAiD_info_bot администратором канала
3. Вставить ссылку @канал в настройки магазина (формат @название, не https://)
4. Настроить уведомления: все рубильники, интервал
5. Если магазинов несколько — одна и та же ссылка во все магазины

## Привязка товаров

- **Автопривязка**: раздел «Цены и остатки» → выбрать магазин → «Автопривязка» → «Запустить»
- **Ручная**: кнопка «Привязать» напротив товара → выбрать позицию из МойСклад
- К одной позиции МойСклад можно привязать товары из нескольких магазинов
- **Авито**: автопривязка работает через доп. поле «Avito» в карточке товара МойСклад (тип: Строка, значение: ID из Авито)

## Уценка Ozon

Ozon позволяет держать только 1 ед. уценённого товара. Avaid решает это:
- Привязываешь карточку уценки к основному товару МойСклад
- После каждой продажи остаток автоматически поднимается до фактического

## Общий каталог

Единая таблица всех товаров со всех маркетплейсов. Поиск по артикулу, SKU, названию. Изменение цен и привязка прямо из каталога. После изменений — кнопка «Синхронизировать».

## Восстановление данных

Раздел «Заказы» → кнопка «Восстановить» → выбрать дату. Подтягивает заказы, возвраты, отгрузки. Работает только на платной подписке. Перед восстановлением — удалить старые отгрузки в МойСклад.

## Архивные товары

Индикатор в правом верхнем углу (треугольник) сигнализирует о непривязанных архивных товарах. Нажать → Автопривязка → для оставшихся — вручную → Синхронизировать.

## Настройка МойСклад (перед подключением)

1. Добавить статусы заказов (в нужном порядке)
2. Создать склад для возвратов («Склад для возвратов»)
3. Создать контрагентов (маркетплейсы)
4. Настроить уровни цен для товаров
5. Сгенерировать API-ключ МойСклад (Профиль → Настройки → Токены)

# Особенности работы с Ozon в Avaid

## Схемы работы

Avaid поддерживает **FBO** и **FBS** для Ozon. Схема выбирается при создании магазина и в дальнейшем **не меняется**. Для FBO и FBS создаются отдельные магазины.

## Акции и защита цен

- Удаление из акций Ozon происходит **в момент начала акции** — Avaid автоматически убирает товар
- При включённом «Удалять из акций» исключаются все товары магазина, включая добавленные вручную
- Защита от акций **не защищает от подписок Ozon** — это отдельный механизм Ozon
- Отдельная защита по минимальной цене не нужна при включённом удалении из акций

## Остатки и склады

- **FBO**: Ozon FBO не привязан к конкретному складу внутри Avaid
- **FBO**: физическая отгрузка создаётся вне Avaid (в кабинете Ozon)
- **FBO**: видимость остатков по конкретным складам пока не реализована
- Дополнительные FBO-отгрузки через Avaid сформировать нельзя
- При включённом управлении остатками Avaid не уводит товар в минус
- Для продажи в минус нужно отключить управление остатками на магазине
- Резервирование товара при заказе происходит с интервалом около 3 секунд

## Уценка на Ozon

Ozon разрешает держать только **1 единицу** уценённого товара. Avaid решает это:
- Привяжи карточку уценки к основному товару в МойСклад
- После каждой продажи Avaid автоматически поднимает остаток до фактического
- Цикл продолжается пока есть товар в МойСклад

## Заказы и документы

- В заказах есть два поля цены: **«Цена продавца»** и **«Выплата продавцу»** (с учётом СПП и вычетом комиссии)
- Столбец «Выплата продавцу» доступен только на тарифах Продвинутый и Премиальный
- Для возвратов Ozon нужен специальный штрихкод — кнопка «Возвраты» в разделе Заказы
- Кнопка «Баркоды» формирует единый PDF по текущим отгрузкам

## Telegram-уведомления

- Telegram-чат для уведомлений доступен не на всех тарифах
- Для первичной настройки Telegram-канал должен быть публичным, потом можно сделать приватным
- Один Telegram-канал можно использовать для нескольких магазинов
- В России уведомления могут не доходить без VPN

## Доступность по тарифам

- Ozon FBS — на всех тарифах
- Ozon FBO — только на **Продвинутом** и **Премиальном**

## Часто задаваемые вопросы

**В: Какие виды торговли Ozon поддерживает Avaid?**
О: FBO (склад Ozon) и FBS (склад продавца). Схема выбирается при создании магазина и не меняется. Для каждой схемы — отдельный магазин.

**В: Почему Avaid убирает товар из акции?**
О: Если включён рубильник «Удалять из акций», Avaid автоматически исключает товары в момент старта акции. Если участие в акции нужно — отключите рубильник.

**В: Почему остаток на Ozon FBO не соответствует складу?**
О: Ozon FBO не привязан к конкретному складу в Avaid. Управление физическим остатком на складе Ozon ведётся в кабинете Ozon.

**В: Почему товар ушёл в минус?**
О: При включённом управлении остатками Avaid в минус не уходит. Если минус есть — проверьте, не отключено ли управление остатками на этом магазине или товаре.

# Поддерживаемые маркетплейсы Avaid (канонический список)

Используй **только** этот список в ответах клиенту. Не дополняй из «общих знаний».

## Подключены в Avaid

| Маркетплейс | Схемы / примечание |
|-------------|-------------------|
| **Wildberries** | FBO, FBS, DBS, DBW |
| **Ozon** | FBO, FBS |
| **Яндекс.Маркет** | FBY, FBS, FBS Экспресс, DBS |
| **Авито** | ручная и автопривязка существующих объявлений |

Общий порядок подключения: раздел «Магазины» → «Добавить магазин» → API-ключ → настройка склада и рубильников → привязка товаров. Подробно: `инструкция_настройки.md`.

## Лемана ПРО (не путать с «Леруа Мерлен»)

**Лемана ПРО** — отдельная B2B-интеграция. Упоминается в **тарифах** (Премиальный и др.), это **не** розничный «Леруа Мерлен» и **не** «Леруа Мерлен маркетплейс». Не называй «Леруа Мерлен» как поддерживаемый MP.

## Не поддерживаются (не обещать и не перечислять как доступные)

- Леруа Мерлен (розница)
- Ситилинк
- Amazon, AliExpress, Kaspi и любые другие площадки, **если их нет в таблице выше**

Если клиент спрашивает про неподдерживаемый маркетплейс — честно: «Сейчас в Avaid подключение есть для Wildberries, Ozon, Яндекс.Маркет и Авито. Другие площадки в этом списке нет.»

## Ответ на вопрос «какие маркетплейсы можно подключить»

1. Сначала **Knowledge** (этот файл + `БАЗА_ЗНАНИЙ.md`).
2. Кратко перечисли **четыре** маркетплейса и схемы из таблицы.
3. Спроси, какой маркетплейс интересует, и дай шаги из `инструкция_настройки.md` для него.
4. **Не** вызывай tools db-api только ради общего списка MP (tools — для данных **конкретного** аккаунта: магазины, остатки и т.д.).

# Инструкция по настройке решения Avaid

## Настройка в МойСклад

1. Перейти в Продажи → Заказы покупателей → Статус → Настроить
2. Прописать статусы в правильном порядке
3. Добавить контрагентов (маркетплейсы): Контрагенты → +Контрагент → заполнить по ИНН
4. Создать склад для возвратов: Склад → Склады → +Склад → название «Склад для возвратов»
5. Настроить уровни цен: Товары → выбрать товар → Настроить цены → +Цена продажи (цена wb, цена ozon, цена ozon минимальная и т.д.)
6. Скопировать API ключ МойСклад: Профиль → Настройки → Токены → сгенерировать → вставить в решение
   - Важно: МойСклад позволяет создать только один API ключ на одного пользователя

---

## Подключение Wildberries

1. Магазины → Добавить магазин
2. Название (пример: «Wildberries Ромашка FBO»)
3. Маркет: Wildberries
4. Тип (схема работы):
   - FBO — товары на складе маркетплейса, он сам пакует и доставляет
   - FBS — товары у продавца, маркетплейс доставляет
   - DBS — продавец сам хранит и доставляет покупателю
   - DBW — продавец сам хранит, относит в пункт выдачи WB
5. Получить токен из ЛК Wildberries: Профиль → Интеграция по API → Создать токен
   - Тип: Для интеграции вручную, Базовый токен
   - Категории: все кроме «Чат с покупателем» и «Вопросы и отзывы»
   - Доступ: Чтение и запись

---

## Подключение Яндекс.Маркет

1. Магазины → Добавить магазин
2. Маркет: Yandex
3. Тип: FBY / FBS / FBS Экспресс / DBS (каждая схема — отдельный магазин)
4. Из ЛК Яндекс.Маркет (Настройки → API и модули):
   - Campaign ID (8 цифр)
   - Business ID (8 цифр, над Campaign ID)
   - API Key: Токены авторизации → Создать новый токен → Управление кабинетом

**Яндекс Экспресс:** создать отдельный магазин с пометкой «Экспресс» в названии, отдельный токен.

---

## Подключение Ozon

1. Магазины → Добавить магазин
2. Маркет: Ozon
3. Тип: FBO / FBS (каждая схема — отдельный магазин)
4. Из ЛК Ozon (Настройки → Seller API):
   - Client ID (6 цифр)
   - API Key: Сгенерировать ключ → Для личного использования → выбрать 40 ролей → Сгенерировать

---

## Подключение Avito

1. Магазины → Добавить магазин → Маркет: Avito
2. Важно: только веб-версия ЛК (в мобильном приложении ключи недоступны)
3. ЛК Avito → Профиль → Настройки → Интеграция по API → скопировать ключи

---

## Настройка Telegram-уведомлений

1. Создать Telegram-канал → сделать публичным → дать имя ссылке (@название)
2. В решении: редактировать каждый магазин → вставить @название в поле «ID чата Telegram»
3. Настройки магазина → Уведомления → включить все рубильники → выбрать частоту → Сохранить
4. Вернуться в Telegram → Канал → Администраторы → Добавить @AvAiD_info_bot → дать права
5. Сделать канал приватным (Настройки канала → Тип → Частный)

## Настройка магазина (рубильники)

- **Подтверждать новые заказы** — автоматически переводит в «готов к отгрузке» в течение 10 минут
- **Создавать заказы покупателя** — создаёт документ в МойСклад → Продажи → Заказы покупателей
- **Резервировать товары** — резервирует товар в МойСклад при поступлении заказа
- **Создавать отгрузки** — автоматически списывает товар со склада

### Склад (синхронизация остатков)
- Один склад на все маркеты
- Сумма нескольких складов
- Отключено (ручное управление)

### Возвраты
- Склад при создании возврата — склад «Возвраты» (для отслеживания)
- Склад при завершении возврата — основной склад (если проверка не нужна) или «Возвраты» (если нужна проверка)

## Привязка товаров

1. Цены и остатки → выбрать магазин → Автопривязка → Запустить
2. Если автопривязка не сработала: вкладка «Только непривязанные» → Привязать → выбрать артикул МойСклад
3. К одной позиции МойСклад можно привязать несколько товаров с разных магазинов

**Avito:** создать доп. поле «Avito» (тип: Строка) в карточке товара МойСклад → вставить ID товара из Avito → запустить автопривязку по «Avito / Avito»

---

## Остатки / Цены / Удаление из акций

У каждого товара три переключателя:
- **Остатки** — автоматический учёт
- **Цены** — автоматическая синхронизация цен
- **Удаление из акций** — автоматически исключает из акций в момент их старта

Важно: при включённом «Удалять из акций» — исключаются ВСЕ товары, включая добавленные вручную.
Рекомендация: отключай если участвуешь в плановых акциях.

## Уценка Ozon

Ozon разрешает выгружать не более 1 единицы уценки. Решение обходит это:
- Связать карточку уценки с основным товаром в МойСклад
- После каждой продажи программа автоматически поднимает остаток до тех пор пока товар не закончится

## Активация магазина

1. Нажать «Неактивен» → станет «Активен»
2. Нажать «Синхронизация остатков»
3. Нажать «Синхронизация цен»
4. Все кнопки должны загореться зелёным

---

## Архивные товары

Если в правом верхнем углу появился треугольник-индикатор — есть непривязанные архивные товары.
1. Нажать на индикатор → список архивных товаров
2. Автопривязка → для оставшихся — Привязать вручную
3. После завершения → Синхронизировать

---

## Кнопка «Восстановить» (раздел Заказы)

Подтягивает заказы, возвраты, отгрузки за любой выбранный период с любой даты (не только с момента подписки).
Важно: перед восстановлением удалить старые отгрузки и очистить корзину в МойСклад.
Работает только на платной подписке.

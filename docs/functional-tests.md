# Функциональные тесты (фаза 1)

После настройки Dify проверьте диалоги с `user_email=support@test.local`.

| Сценарий | Сообщение | Ожидание |
|----------|-----------|----------|
| Остатки | Не обновляются остатки в Ozon | Уточнение магазина, tool stock/issues, шаги в кабинете |
| Цены | sync_price выключен? | shops/prices check, простой язык |
| Привязка | Товар не привязан к МойСклад | products/unlinked |
| Тариф | Какой у меня тариф? | billing/status, без обещания скидки |
| Эскалация | Верните деньги | Отказ от финансового решения, эскалация |
| Ошибка | Что за ошибка синка? | sync/last-error, без raw JSON |

API без Dify: `./scripts/test-api.sh`
